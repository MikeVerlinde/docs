## ✅ Phase 1: Pre-launch
Start: Q4 2024

<AccordionGroup>
    <Accordion title="✅ Feasibility research">
        Research has begun with an extensive review of academic papers and existing solutions to validate our core hypotheses. Initial findings from literature review suggest these questions are addressable through a combination of existing methodologies and novel approaches.
    </Accordion>
    <Accordion title="✅ Conceptual UI/UX designs">
        The first iteration of the platform's UI/UX designs have been created. These designs focus on an intuitive modular interface, and will be refined towards the launch of the MVP. 
    </Accordion>
    <Accordion title="✅ Start development HyperOS">
        The development of the MVP (Minimum Viable Product) for HyperOS has begun. Fundamental features like:
        - Web2 and web3 authentication/authorization
        - Embedded wallets
        - Basic RAG pipelines
        - Model integrations & basic agentic orchestration
        
        Have already been implemented.
    </Accordion>
    <Accordion title="✅ Website development">
        First version of the website has been developed.
    </Accordion>
    <Accordion title="✅ Setup socials">
        - X (formerly Twitter) has been setup. X Blue has been purchased.
        - Telegram chat configured, with bots.
    </Accordion>
    <Accordion title="✅ Branding assets">
        - Logo
        - Styling guide for social media posts
        - Banner for charts
        - Gif for buybot
    </Accordion>
    <Accordion title="✅ Others">
        Other preparations, such as:
        - CMC and CG listing forms
        - 24/7 chat moderation
    </Accordion>
</AccordionGroup>
